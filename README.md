# Test ERC20 token for Ownera Hedera adaptor testing

This repo was used to deploy a test ERC20 contract to hedera testnet for Ownera to test their Hedera Addaptor against

The contract is a openzeppelin ERC20 contract with the Minter role added using the Accesscontrol module.

Admin and minter account addresses can be found in [text](ignition/parameters.hederatestnet.json)
